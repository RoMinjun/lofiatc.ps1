# lofiatc.ps1
lofiatc alternative using PowerShell and VLC

# Gettings Started

## Clone repo locally
git clone https://github.com/RoMinjun/lofiatc.ps1.git

## Run script
```powershell
.\lofiatc.ps1
```

> [!TIP]
Check the possibilities of script using `Get-Help` 
```powershell
Get-Help .\lofiatc.ps1 -Full
```
